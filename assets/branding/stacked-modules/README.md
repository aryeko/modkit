## modkit branding: Stacked Modules

This directory contains the **Stacked Modules** branding assets (the “M” mark).

### Files

- **`modkit-social-preview.png`**: Social preview banner (source, as exported).
- **`modkit-social-preview-1280x640.png`**: Resized PNG (**1280×640**).
- **`modkit-social-preview-1280x640.jpg`**: Recommended for GitHub **Social preview** upload (**1280×640**, usually smaller than PNG).
- **`modkit-square.png`**: Square logo (source, as exported).
- **`modkit-square-512.png`**: Square logo (**512×512**).
- **`modkit-square-512.jpg`**: Optional JPEG avatar (smaller alternative).
